## Requirements to run this script:
```
nodejs 14.xxx
npm 6.xxx
```

## Install for Windows/Vps/Rdp
```
git clone https://github.com/rollebot/rollebot-openowa
cd rollebot-openwa
node .
```
## Install for Linux/Ubuntu
```
apt sudo install git
git clone https://github.com/rollebot/rollebot-openowa
node .
```
